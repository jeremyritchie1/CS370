# CS370

For this project I was given a working maze environment in TreasureEnvironment.py, an experience replay class in GameExperience.py, a pre-built neural network, and a skeleton training loop in TreasureHuntGame.ipynb. What I wrote myself was the qtrain function, which is the heart of the deep Q-learning algorithm. That included epsilon-greedy action selection, the experience replay loop, a target network that syncs on a fixed schedule to keep training stable, epsilon decay tied to a sliding win rate, and early stopping once the agent hits 100% across all free cells. The agent reached that threshold at epoch 287, which was the clearest sign that converting raw trial and error into a reliable policy through reinforcement learning actually worked.
Computer scientists build systems that solve problems too large, too fast, or too complex for humans to handle manually, and that work matters because those systems increasingly shape how people access information and make decisions. This project is a small version of that idea: instead of hand-coding a route through the maze, I defined a reward signal and let the agent find the path on its own. When I approach a problem that way I try to get the smallest testable piece working first, then build outward. With the pirate agent that meant getting a single episode to run cleanly before worrying about whether the agent was actually learning, then tuning the epsilon schedule and target network update frequency once the mechanics were solid.
The ethical side of reinforcement learning is easy to underestimate when the task is navigating a grid, but the same techniques scale to systems that affect real people. That means being honest about what a model can and cannot do, not overstating its reliability, and making sure there is enough transparency that someone can audit why the system made a particular choice. A model that gets the right answer most of the time is not the same thing as a trustworthy system, and that distinction matters both to the end user who depends on it and to the organization responsible for it.
